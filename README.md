# Verdict — Forensic Medicine Exam Prep

A self-contained web app for studying forensic medicine, built for Thai medical licensing exam preparation.

## Features
- 8 study modules with full lesson content
- 350+ practice questions across all topics
- Mock exam mode with timer
- XP and rank progression system
- Weak topic detection and targeted practice
- Progress tracking saved locally in your browser
